
THIS WEBSITE IS AN EXACT HTML VERSION OF THE SARAS 7.0 MANDATORY DISCLOSURE PDF.

HOW TO UPDATE:

1. DOCUMENT LINKS (Section B & C)
Open: config.js
Each entry in sectionB and sectionC has a URL as the 3rd item.
Replace "https://sites.google.com/view/dugagss/home" with the actual URL for each document.

2. YOUTUBE LINK
Open: config.js
Find: youtubeVideo: "https://youtu.be/wDchsz8nmbo"
Replace ONLY the link inside the quotes with your actual YouTube video link.

FILES:
- index.html   → Main page (structure & content)
- style.css    → All styling
- script.js    → Populates Section B, C tables and YouTube link
- config.js    → All configurable links and URLs (edit this file)
